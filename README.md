# HTML BOX 🎯

一款基于 PyQt5 开发的深色主题 HTML 编辑器，提供实时预览、语法高亮和代码导出功能。

## ✨ 功能特色

### 🎨 编辑器功能
- **实时预览** - 边写代码边查看效果
- **语法高亮** - 支持 HTML、CSS、JavaScript 语法高亮
- **深色主题** - 护眼暗色界面，长时间编码不疲劳
- **智能提示** - 自动补全和代码格式化

### 📁 文件管理
- **新建/打开/保存** - 完整的文件操作功能
- **多格式导出** - 支持单个 HTML、完整项目 ZIP、压缩版 HTML
- **项目模板** - 导出标准项目结构（HTML + CSS + JS）

### ⚙️ 自定义设置
- **字体设置** - 自定义编辑器字体和大小
- **预览延迟** - 调整实时预览响应速度
- **主题配置** - 个性化界面外观

### 🎮 趣味功能
- **版本历史** - 完整的更新日志记录
- **小彩蛋**   -自己发掘吧（WvW）
## 🚀 快速开始

### 环境要求
- Python 3.6+
- Pyside6
- 其他依赖包

### 安装步骤

1. **克隆项目**
   ```bash
   git clone https://github.com/Sevunne/html-box.git
   cd html-box
   ```
2. **一键安装全部依赖**
   ```bash
   pip install pyside6 pyside6-qtwebengine
   ```
## 📖 使用说明
1. 顶部工具栏左侧放置新建、打开、保存、导出等基础文件功能，右侧通过弹性空白分割，集中放置模式切换与工程操作按钮。
2. 点击「📝 专业代码模式」切换至代码编辑面板，手动编写 HTML / CSS / JS 代码，编辑器自带语法高亮。
3. 点击「🧩 积木节点模式」切换至可视化积木面板，拖拽组件快速搭建网页。
4. 模式切换内置缓存机制，手动编写的代码不会被积木内容覆盖丢失，两种编辑模式数据双向互通。
5. 支持 `.htbx` 专属工程格式，可保存整套积木布局与代码内容，下次打开完整恢复工程。
6. 顶部「预览」按钮可手动刷新网页，也支持自动定时预览，预览延迟可在设置面板自定义。
7. 全功能配备快捷键，底部状态栏实时展示当前编辑模式与程序状态。
